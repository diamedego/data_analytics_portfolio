# Wind Farm Performance Analysis – Aerowind

IMAGEN

## Project Overview

This project analyzes the operational and economic performance of the **Aerowind Wind Farm** using SQL and Power BI.
The goal is to transform raw operational, climatic and maintenance data into a structured analytical model that allows monitoring turbine performance, identifying operational issues and evaluating the profitability of wind energy generation.

---

## Business Problem

Aerowind operates a wind farm composed of multiple turbines installed between 2023 and 2025.
However, the company lacks a unified system to analyze operational performance.

Key questions addressed in this project:

* Which turbines generate the most energy?
* How do climatic conditions affect production?
* What is the impact of maintenance events on downtime?
* What is the economic performance of the wind farm?

The objective was to create a **data model and analytical dashboard** to support operational monitoring and decision-making.

---

## Dataset

The analysis integrates six datasets:

| Dataset            | Description                                    |
| ------------------ | ---------------------------------------------- |
| production         | Energy generated by each turbine               |
| maintenance        | Maintenance events and associated costs        |
| climate            | Wind speed, temperature, humidity and pressure |
| turbines           | Technical specifications of turbines           |
| prices             | Energy market prices                           |
| theoretical_curves | Expected power generation based on wind speed  |

All datasets are available in the **/data** folder.

Data preparation was performed using SQL.

---

## Data Model

After cleaning, the datasets were integrated into a relational model used for analytical exploration in Power BI.

Main relationships:

* Production → Turbines
* Maintenance → Turbines
* Climate → Production (date)
* Prices → Production (date)

This structure enables analysis across turbines, time periods and environmental conditions.

---

## Dashboard

The Power BI dashboard provides four analytical views:

**Energy Performance**

* Energy generation by turbine
* Generation trends over time

**Operational Reliability**

* Maintenance events
* Turbine downtime analysis

**Wind Resource Analysis**

* Wind speed distribution
* Wind conditions vs energy output

**Economic Performance**

* Revenue estimation
* Production vs market price

The Power BI file is available in:

```
/powerbi/aerowind_dashboard.pbix
```

---

## Key Insights

Some relevant findings from the analysis include:

* Significant differences in performance between turbine models
* Recurring downtime patterns in specific turbines
* Wind conditions explain part of production variability
* Newer turbine models show higher generation efficiency

These insights can support operational optimization and maintenance planning.
